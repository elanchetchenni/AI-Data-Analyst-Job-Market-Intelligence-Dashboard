# AI & Data Analyst Job Market Intelligence Dashboard

## Project Overview
This Power BI dashboard analyzes current job market trends for AI and Data Analyst roles.

## Objectives
- Analyze in-demand skills
- Track salary trends
- Compare job roles
- Identify hiring locations

## Tools Used
- Power BI
- SQL
- Excel
- Python

## Dashboard Features
- KPI Cards
- Interactive Filters
- Skill Demand Analysis
- Salary Insights
- Location Analysis
